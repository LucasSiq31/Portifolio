Este repositório contém o código-fonte do meu portfólio pessoal. Uma página desenvolvida para centralizar meus projetos, habilidades técnicas e trajetórias no mundo do desenvolvimento de software.

---

### 📑 Seções do Projeto

O site foi estruturado para oferecer uma navegação rápida e informativa:

* **Sobre Mim:** Uma breve apresentação sobre minha jornada acadêmica em Engenharia de Software e meus objetivos profissionais.
* **Stack Tecnológica:** Exibição das linguagens, frameworks e ferramentas que domino e utilizo em meus projetos.
* **Projetos:** Showcase das principais aplicações desenvolvidas (Web, Mobile e Lógica).
* **Contato:** Canais diretos para networking, incluindo GitHub, LinkedIn e E-mail.

### 🛠️ Tecnologias Utilizadas

Para garantir uma interface leve, rápida e elegante, utilizei:

* **HTML5:** Estruturação semântica de todo o conteúdo.
* **CSS3:** Estilização personalizada, Layouts com Flexbox/Grid e design responsivo para todos os dispositivos.

---

### 🎨 Diferenciais do Projeto
- **Design Clean:** Foco total na legibilidade e na experiência do usuário (UX).
- **Mobile First:** Totalmente adaptado para leitura em smartphones.
- **Performance:** Carregamento instantâneo por ser uma aplicação estática otimizada.
